JSONCoder

public struct JSONCoder

Utility for encoding serializable values as JSON.

  • encode(_:)

    Encodes a serializable expression as JSON data.

    Throws

    An error if the data could not be converted to JSON. This may happen if a value does not implement Serializable correctly and generates a serializable expression which does contain types other than primitive types (int, float, double, etc…), strings, arrays or dictionaries.

    Declaration

    Swift

    public static func encode(_ value: Serializable) throws -> Data

    Parameters

    value

    Value which should be serialized

    Return Value

    JSON data generated by serializing the given value

  • decode(_:)

    Deserializes JSON data into an expression of the required type.

    Throws

    An error if the JSON data is syntactically incorrect, contains invalid values or is incomplete.
