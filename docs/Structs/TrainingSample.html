A training sample which also provides an expected output

  • init(values:expected:)

    Initializes a training sample which can be used to train a neural network using backpropagation.

    The training sample stores an input aswell as the output which is expected for the given input.

    The input value must match the network input dimensions. The output value must match the network output dimensions.

    Parameters

    Input value for the network.
    expected

    Expected output for the given input
  • init(values:outputCount:targetIndex:baseValue:hotValue:)

    Initializes a training sample from an input and an expected output which is a one-hot-vector.

    A one-hot-vector contains zeros at all indices except one hot index.

    The output will be a matrix with a width and height of one and a depth of outputCount.
