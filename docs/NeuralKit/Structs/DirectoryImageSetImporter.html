  • import(from:baseOutputValue:hotOutputValue:includeFile:label:pixelDataOfImage:)

    Imports all training samples from the given directory.

    The training samples must be stored as images in either PNG or JPG format.

    Throws

    Throws an error if the contents of the directory, its subdirectories or files could not be read.

    Declaration

    Swift

    public static func `import`(
		from directory: URL,
		baseOutputValue: Float = 0,
		hotOutputValue: Float = 1,
		includeFile: @escaping (URL) throws -> Bool = {_ in true},
		label: @escaping (URL) throws -> String = {$0.path},
		pixelDataOfImage: @escaping (CGImage) -> Matrix3? = {$0.greyscaleMatrix}
	) throws -> [(TrainingSample, Label)]

    Parameters

    directory

    Directory to import samples from
    baseOutputValue

    Output in the training samples used if an output does not correspond to the label of a sample
    hotOutputValue

    Output in the training samples used if an output represents the label of a sample
    includeFile

    Decides if a file should be imported. True by default.
    label

    Label for a sample. By default, the path to the file excluding the filename itself will be used. Samples with equal labels will be grouped together.
    pixelDataOfImage

    Retrieves the pixel data of an image and converts it to a Matrix3. By default, the greyscale image of the matrix will be used.

    Return Value

    List of samples and their corresponding label

  • Label

    Label of a training sample. The label will be equal to the path of the folder it is contained in
