  • init(inputSize:outputSize:)

    Creates a new Max Pooling layer with the given input and output size. Scaling factors are determined automatically from the input and output size.

    A pooling layer can be used to reduce the size of the forwarded volume and thereby reducing the computational cost of subsequent layers. A pooling layer can also increase the size of the receptive field of subsequent convolution layers and prevent overfitting.
