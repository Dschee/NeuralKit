The following functions are available globally.

  • +(_:_:)

    Appends the right CGImageFilter to the left CGImageFilter.

    When using the returned CGImageFilter, the filter, which was passed left to the ’+’-operator will be evaluated first. Then the filter passed on the right will be evaluated with the result of the left filter.

    The ’+’-operator for CGImageFilters is associative: (a + b) + c = a + (b + c)

    The ’+’-operator is non-commutative: a + b ≠ b + a

    Declaration

    Swift

    public func + (left: @escaping CGImageFilter, right: @escaping CGImageFilter) -> CGImageFilter

    Parameters

    left

    CGImageFilter which should be evaluated first
    right

    CGImageFilter which should be evaluated second

    Return Value

    A newly created CGImageFilter-function consisting of the two filter functions.

  • CGImageFilterAppendFilter(filter:toFilter:)

    Appends the first CGImageFilter to the second CGImageFilter.

    When using the returned CGImageFilter, the filter, which was passed second to the ’+’-operator will be evaluated first. Then the filter passed second will be evaluated with the result of the left filter.

    Declaration

    Swift

    public func CGImageFilterAppendFilter(filter second: @escaping CGImageFilter, toFilter first: @escaping CGImageFilter) -> CGImageFilter

    Parameters

    second

    CGImageFilter which should be evaluated last
    first

    CGImageFilter which should be evaluated first

    Return Value

    A newly created CGImageFilter-function consisting of the two filter functions.
