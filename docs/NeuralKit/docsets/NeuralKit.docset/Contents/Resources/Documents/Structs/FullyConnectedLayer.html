  • init(weights:)

    Initializes a fully connected neural layer using the given weight matrix, its activation function and derivative

    The value at column n and row m of the weight matrix corresponds to the weight of the nth neuron towards the mth neuron of the next layer.

    Note: The input size of the layer will be one neuron smaller than the width of the weight matrix as the weight matrix will also store bias values of the layer

    public init(weights: Matrix)

    Parameters
