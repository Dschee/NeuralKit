NeuralLayerEncoder

public struct NeuralLayerEncoder

Utility for encoding neural layers.

Encodes the type of a layer next to it, so it can be correctly restored during deserialization

  • serialize(_:)

    Serializes a layer and also stores its type for deserialization.

    If the layer is not included in the set of default layers (fully connected layer, convolutional layer, pooling layer) it must be registered before deserialization using the NeuralLayerEncoder.registerLayerType(:) method.

    Deserializes a layer which was encoded using the encode function.
