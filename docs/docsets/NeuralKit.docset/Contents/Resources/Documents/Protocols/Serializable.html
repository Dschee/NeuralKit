  • serialized()

    Serializes an existing instance of a serializable type into an expression which can be used by (NS)JSONSerialization.

    Allowed types are primitive types (int, float, double, etc…), strings, arrays and dictionaries.

    If another value should be stored in the serialized representation, it must implement the Serializable protocol and it must not be stored directly but using its serialized representation.
